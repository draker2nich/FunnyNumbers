# FunnyNumbers — Числовая викторина

**FunnyNumbers** — увлекательное мобильное приложение для Android, превращающее математику в интересную викторину. Отлично подходит для тех, кто хочет развивать навыки работы с числами и весело проводить время.

## Установка

### Разработка

1. Клонируйте репозиторий:

```bash
git clone https://github.com/draker2nich/FunnyNumbers.git
```

2. Откройте проект в Android Studio
3. Запустите на эмуляторе или реальном устройстве

### Установка APK

Скачайте последний релиз из раздела [Releases](https://github.com/draker2nich/FunnyNumbers/releases) и установите на устройство Android.
